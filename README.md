# gogainda/homebrew-tap

Homebrew formulae for [gogainda](https://github.com/gogainda)'s tools.

## Install

```bash
brew install gogainda/tap/mdview-cli
```

or

```bash
brew tap gogainda/tap
brew install mdview-cli
```

## Formulae

| Formula | Description |
|---|---|
| `mdview-cli` | Preview Markdown files with [mdview.io](https://mdview.io) from the terminal (`mdv`) |
